Tally System 종합 설명서

ATEM1000-12Pro 모델을 기준으로, 블랙매직 ATEM 스위처와 연동하여 사용하는 방법을 안내합니다.

✅ 구성품 세트 내용 (완제품 풀 세트)

  • 탈리 인터페이스 본체 1개
  • 무선 탈리 수신기 12개 (채널별 번호 표시)
  • USB 충전 스테이션 (8포트)
  • USB Type-C 케이블 (충전용 다수)
  • 네트워크 케이블 (RJ45 케이블)
  • USB 전원 케이블 (인터페이스 전원 연결용)
  • 무선 안테나 (인터페이스 및 수신기용)
  • 설치용 마운트 부품 및 예비용 공구 포함

📌 Tally Interface (본체) 상세 기능

전면부 (Front)

  • ① 탈리 상태 표시등: 총 24개 LED. 위쪽은 PGM(방송중-빨강), 아래쪽은 PVW(대기중-초록)를 표시합니다.
  • ② 리셋(Rst) 홀: 길게 눌러 장비 IP를 `192.168.10.200`으로 초기화할 때 사용합니다. (일반 사용 시 누르지 않음)
  • ③ SYS 표시등: 전원 인가 10초 후부터 깜박이며, 스위처와 정상적으로 통신 중임을 나타냅니다.
  • ④ PWR 전원 표시등: 전원이 연결되면 항상 켜져 있습니다.
  • ⑤ 전면 RJ45 포트: LED가 내장된 기가비트 포트. ATEM 스위처, 컴퓨터, 컨트롤 패널 등을 연결합니다.

후면부 (Back)

  • ⑥ 5V USB 전원 포트: 1A 출력 기준 전원 공급. 1개는 주 전원, 1개는 예비 전원용입니다. (vMix 사용 시 전용 케이블 사용)
  • ⑦ DB25 포트: 24채널 유선 탤리 출력을 지원하는 단자. 외부 릴레이 장치 등과 연결 가능하며, 13번 핀은 독립 접지선입니다.
  • ⑧ 후면 RJ45 포트: 전면 포트와 동일한 기능을 하는 추가 네트워크 포트입니다.
  • ⑨ 무선 확장 인터페이스: 옵션 품목인 Tally Pro 무선 수신기를 위한 안테나를 연결하는 포트입니다.

📌 무선 수신기 (Tally RX Pro) 상세 기능

간단 요약

항목역할 및 용도설정 방법 예시
MODE 버튼 수신기의 동작 방식 설정 (일반/테스트/페어링) 버튼을 눌러 모드 변경
ID 로터리 스위치 각 수신기에 고유한 채널 번호를 부여하여 송신기와 연결 카메라 번호와 같은 숫자로 설정
Tally OUT 수신한 탤리 신호를 다른 외부 장치(LED, 조명)에 출력 외부 LED나 조명 시스템 연결

🎬 설치 및 설정 방법 (ATEM 스위처 기준)

기본 IP 설정

이 탤리 시스템은 ATEM 스위처의 IP가 **`192.168.10.240`**으로 설정된 것을 기준으로 작동합니다. 따라서, ATEM Software Control 프로그램을 통해 스위처의 IP를 직접 설정해야 합니다.

간략한 사용법 요약

  • ATEM 스위처와 인터페이스 간 IP 설정을 맞추고 네트워크로 연결합니다.
  • 수신기 채널(ID)을 카메라 번호와 동일하게 설정 후 전원을 켭니다.
  • 카메라에 장착 후 스위처에서 PGM/PVW 선택 시 탤리 상태가 실시간으로 표시됩니다.

🔋 배터리 및 실사용 가능 시간

배터리 용량은 1200mAh (리튬폴리머 3.7V)입니다.

실사용 가능 시간 (일반 기준)

사용 환경사용 가능 시간 (완충 기준)
LED 밝기 최대 + 빈번한 신호 수신약 8~10시간
일반적인 방송 환경 (평균적인 사용)약 10~12시간
LED 밝기 최소 + 신호 수신 빈도 적음약 12시간 이상

⚠️ 실사용 팁

  • 촬영 전에는 항상 완전 충전을 권장합니다.
  • 장시간 방송의 경우 여분의 수신기 또는 보조 배터리를 준비하면 더욱 안정적입니다.
  • LED 밝기를 낮추면 배터리 사용 시간이 더 길어집니다.

🚨 중요 주의사항

  • ATEM 스위처 IP를 **`192.168.10.240`**으로 설정하거나, 전용 소프트웨어로 IP를 통일해야 정상 작동합니다.
  • DB25 커넥터 사용 시, **13번 핀에 반드시 별도의 접지선을 연결**하여 장비를 보호하세요.
  • 수신기는 촬영 전 **배터리를 충분히 충전**하고 상태를 확인하세요.
